2-font-style ( 字体风格 )

italic： 浏览器会显示一个斜体的字体样式。
oblique： 浏览器会显示一个倾斜的字体样式。
font-style 属性定义字体的风格,设置使用斜体、倾斜或正常字体。斜体字体通常定义为字体系列中的一个单独的字体

3-font-variant ( 字体变形 )

small-caps：浏览器Love you forever会显示小型大写字母的字体。
normal (默认值)：浏览器会显示一个标准的字体。

4-font-weight ( 字体加粗 )

bold：定义粗体字符。
bolder：定义更粗的字符。
lighter： 定义更细的字符。
100~900： 定义由粗到细的字符。400 等同于 normal，而 700 等同于 bold。(该属性值只有整百)

5-简写形式：font

            font 简写属性在一个声明中设置所有字体属性。此属性可以设置："line-height"指定行间距。
            可以按顺序设置如下属性：
    
            1. font-style
            2. font-variant
            3. font-weight
            4. font-size/line-height
            5. font-family
